# PWA Icons Directory

This directory contains the Progressive Web App icons for RADIUS.

## Required Icons

The following icon sizes are needed for full PWA support:

- icon-16x16.png
- icon-32x32.png
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Creating Icons

1. Start with a high-resolution image (at least 512x512px)
2. Ensure the design works well at small sizes
3. Use a transparent background or the app's theme color
4. Consider maskable icons for better Android integration

## Tools for Icon Generation

- [PWA Builder](https://www.pwabuilder.com/imageGenerator) - Free online generator
- [Favicon.io](https://favicon.io/) - Multiple format generator
- [RealFaviconGenerator](https://realfavicongenerator.net/) - Comprehensive generator
